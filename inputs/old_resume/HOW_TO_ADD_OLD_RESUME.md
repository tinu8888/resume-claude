# Drop an old resume here (optional)

If you have a previous resume, put it in this folder (`inputs/old_resume/`). PDF, Word, or text all work.

`/start` uses it as extra reference alongside your LinkedIn profile, to catch details, numbers, and phrasing that LinkedIn might miss.

No old resume? That's fine. `/start` works from LinkedIn alone.

You can delete this file. It's just a guide.
